Get started

Deploy

Static page, live runtime, or verified local package.

JTML can emit plain HTML for static hosting, run a tiny server for reactive event handling, and produce a local release artifact for controlled environments.

Static

jtml transpile examples/html_syntax.jtml -o dist/index.html

Host the output anywhere that serves static files.

Live

jtml serve examples/starter_counter.jtml --port 8000 --watch

The server provides HTTP, WebSocket updates, and an HTTP event fallback.

Studio

jtml demo --port 8000

Open the local authoring surface with Run, Lint, Format, Export HTML, diagnostics, versioned local Save/history, snippets, and preview.

Install

cmake -S . -B build -DJTML_BUILD_PYTHON=OFF
cmake --build build --target jtml_cli
cmake --install build --prefix ~/.local

Put `~/.local/bin` on your `PATH`, then run `jtml --version`.

Package

scripts/package_cli.sh dist/release

The release bundle includes docs, examples, tutorial, editor assets, website source, manifest, and checksums.

Predeploy Verify

scripts/verify_all.sh

Build tests, smoke examples, generate the static site, and produce the release archive before publication.
