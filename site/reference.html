Reference

The language in one page.

Friendly JTML 2 is the canonical authoring syntax. The full keyword index and versioned reference ship with every site artifact as language-reference.md.

Canonical Syntax

jtml 2

page class "profile"
  h1 "Hello"

Use Friendly `jtml 2` for new apps, tutorials, Studio, docs, examples, and AI-authored code. The Classic backend remains available for migration and artifacts.

State

let count = 0
const limit = 10
get doubled = count * 2
count += 1

`let` is mutable, `const` is immutable, and `get` tracks dependencies through the semantic graph.

Actions

when add
  count += 1

button "Add one" click add

Friendly event sugar calls JTML actions without user JavaScript.

Fetch

let users = fetch "/api/users" stale keep retry 2

page
  if users.loading
    text "Loading..."
  for user in users.data
    text user.name

Fetch values expose loading, data, error, stale, retry, and request metadata.

Routes

route "/" as Home
route "/user/:id" as User
route "*" as NotFound

link "Open user" to "/user/42"

Routes are hash-based today, with params, layouts, guards, active links, lazy route loads, and redirects.

Semantic UI

theme
  color primary "#2563eb"

panel title "Usage" pad lg shadow md
  grid cols 4 gap md
    metric "Users" total "Active" tone good

Semantic primitives lower to ordinary HTML and generated classes while remaining visible to tooling.

Media And Graphics

file "Choose image" accept "image/*" into selected
image src selected.preview alt selected.name
chart bar data rows by label value total
scene3d "Product" scene model into sceneState

JTML includes media bindings, SVG-first chart/shape aliases, and 3D canvas mounts with host renderer hooks.

Interop

extern notify from "host.notify"
html raw "<host-widget></host-widget>"
css raw
  host-widget { display: block; }

Escape hatches are explicit and reported by lint/semantic tooling for production review.
