Reference

The language in one page.

JTML keeps HTML's element model, then adds Friendly state, derived values, actions, components, modules, fetches, routes, stores, effects, and runtime tooling.

Elements

jtml 2

page class "profile"
  h1 "Hello"

Friendly `jtml 2` is the canonical syntax. Classic `@tag` and `element` forms remain supported for compatibility.

State

let count = 0
const limit = 10
get doubled = count * 2
count += 1

`let` is mutable, `const` is immutable, `get` tracks dependencies, and compound assignment is supported.

Conditional Values

get label = saved ? "Saved" : "Unsaved"

button disabled saved
  show saved ? "Done" : "Save"

Use `condition ? valueA : valueB` anywhere an expression is accepted.

Events

when add
  count += 1

button "Add one" click add

Friendly event sugar calls JTML actions without user JavaScript.

HTML Parity

label for "email" aria-label "Email"
  show "Email"

input "Email" id "email" required

my-widget data-mode "compact"
  show "Custom element"

Hyphenated attributes, keyword-shaped names, custom elements, and void elements are supported.
