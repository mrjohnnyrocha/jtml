# 7. Putting It Together

A small note-keeping app composed from the beginner concepts: `let`, `get`, `when`, `into`, `click`, and string building.

This is the point where JTML starts feeling like an application language instead of a template language. The next lessons take the same mental model into production-shaped features: async data, shared stores, routes, reusable components, Classic compatibility, and CLI tooling.
